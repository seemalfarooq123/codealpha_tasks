## ✅ Task 1: Language Translation Tool

### 🔧 Technologies Used
- Python
- Flask (Backend)
- HTML, CSS (Frontend)
- Google Translate API (googletrans)
- gTTS (Text-to-Speech)

---

### 🚀 Features
- 🌐 Multi-language text translation
- 🔄 Automatic language detection
- 🎯 User-friendly interface
- 🔊 Text-to-speech (audio output)
- 📋 Copy translated text
- 🎨 Modern UI (Glassmorphism design)

---

### 🖥️ How It Works
1. User enters text
2. Selects source and target language
3. Clicks "Translate"
4. App sends request to translation API
5. Displays translated text
6. Generates and plays audio output

---

### 📁 Project Structure
Task1_Language_Translator/
│── app.py
│── templates/
│ └── index.html
│── static/
│ └── output.mp3


---

### ▶️ How to Run

1. Install dependencies:
   pip install flask googletrans==4.0.0-rc1 gtts


2. Run the app:

python app.py


3. Open in browser:

http://127.0.0.1:5000

✅ Task 2: Chatbot for FAQs
🔧 Technologies Used
Python
Flask
HTML, CSS
Scikit-learn (TF-IDF, Cosine Similarity)
NLTK (Text preprocessing)
🚀 Features
💬 Interactive chatbot interface
🧠 NLP-based text preprocessing
🔍 FAQ matching using cosine similarity
⚡ Real-time response generation
🎨 Modern UI design
🖥️ How It Works
User enters a query
Text is preprocessed (lowercase, punctuation removal)
TF-IDF vectorization is applied
Cosine similarity finds the closest FAQ
Best matching answer is returned

│   └── index.html
▶️ How to Run
